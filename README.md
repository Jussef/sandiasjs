## Extension Settings

Incluye los console de colores.

Por ejemplo:

`console.log("console.log('%c $1', 'font-size: 100%; color: #00AEFF; font-weight: 700;');")`

* `myExtension.enable`: enable/disable this extension
* `myExtension.thing`: set to `blah` to do something
